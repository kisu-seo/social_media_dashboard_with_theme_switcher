# Front-end Style Guide

## Layout

The designs were created to the following widths:

- Mobile: 375px
- Tablet: 768px
- Desktop: 1024px


## Colors (Updated from Design System Image)

- **White**: Hex `#FFFFFF` | RGB `255, 255, 255` | HSL `hsl(0, 100%, 100%)`
- **Black**: Hex `#000000` | RGB `0, 0, 0` | HSL `hsl(0, 0%, 0%)`

## Typography (Updated from Design System Image)

### Font Family
- **Primary Font**: [Inter](https://fonts.google.com/specimen/Inter)
- **Weights**: Regular (400), Bold (700)

### Text Presets

| Preset | Font Size | Line Height | Letter Spacing | Font Weight |

| **Preset 1** | 56px | 85% | -2px | Bold |
| **Preset 2** | 32px | 125% | 0px | Bold |
| **Preset 3** | 28px | 125% | 0px | Bold |
| **Preset 4** | 24px | 125% | 0px | Bold |
| **Preset 5** | 14px | 125% | 0px | Bold |
| **Preset 6 (Bold)** | 12px | 125% | 0px | Bold |
| **Preset 6 (Regular)** | 12px | 125% | 5px | Regular |

*(Note: Preset 6 Regular with 5px letter-spacing is typically used for uppercase subheadings like "FOLLOWERS" or "OVERVIEW - TODAY")*
